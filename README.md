# ALU Connect

ALU Connect is a Flutter mobile application designed to help students connect, communicate, and engage within the African Leadership University community. The application provides a seamless user experience with secure authentication and cloud-based data storage powered by Firebase.

## 📱 Features

- User registration and login using Firebase Authentication
- Secure user authentication
- Real-time data storage with Cloud Firestore
- Responsive and intuitive user interface
- Form validation
- User profile management
- Cross-platform support (Android, iOS, macOS)

## 🛠️ Technologies Used

- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Provider (State Management)
- Material Design

## 🚀 Getting Started

### Prerequisites

Before running the project, ensure you have:

- Flutter SDK installed
- Dart SDK
- Android Studio or VS Code
- Firebase CLI (optional)
- A configured Firebase project

### Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/mobile-dev-formative-2.git
```

2. Navigate into the project

```bash
cd mobile-dev-formative-2
```

3. Install dependencies

```bash
flutter pub get
```

4. Configure Firebase

Add the Firebase configuration files:

- Android: `android/app/google-services.json`
- iOS/macOS: `GoogleService-Info.plist`

5. Run the application

```bash
flutter run
```

## 📚 Challenges Encountered

During development, several challenges were encountered, including:

- Configuring Firebase for macOS.
- Resolving Swift Package Manager dependency issues.
- Managing authentication flow.
- Handling asynchronous Firebase operations.

These challenges were resolved through proper dependency management, cache cleaning, and package resolution.
